American Sign Language Using Artificial Intelligence (ASLAI)

ASLAI combines computer vision and machine learning to create an artificial intelligence model that recognizes the American sign Language (ASL) alphabet. With the use of convolutional neural networks, we were able to train a model given a dataset of hand gestures to predict the letter the user was signing. The model was then imported to a webpage we created to allow users to have a more apealing and easier to navigate UI.

Music Genre Recognition Using Spectrograms

With the use of machine learning, this project aims to create and train an artificial intelligence model that can accurately predict the genre of a song given the correpsonding song's spectrogram. An existing dataset on GitHub was used for this project, and the model was trained using both Google Colab and Jupyter Notebook. As of the last update to the project, the model has ~60% accuracy. A report with my findings, confusion matricies, and list of short comings as well as possible improvments that can be made to increase the model accuracy are available in the project repository.
